# Judgment Transparency Principle

This repository contains the original and authoritative formulation of the  
**Judgment Transparency Principle**.

本リポジトリは、**判断透明性原則（Judgment Transparency Principle）**の  
原典かつ正本を格納するものである。

---

## Document Order of Authority

The following documents define the principle, in descending order of authority.  
In the event of any conflict, higher-order documents take precedence.

1. **JTP.md**  
   Original text (English)

2. **JTP_ja.md**  
   原典（日本語）

All other files in this repository are non-normative and exist solely to  
support reference, attribution, or distribution.

---

## Status

This repository defines a principle.  
It is **not** an implementation guide.

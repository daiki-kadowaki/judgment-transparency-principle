This repository does not define the Judgment Transparency Principle (JTP) itself.

The canonical formulation of JTP is published as an academic paper on SSRN.
This repository serves as a secondary reference providing
implementation notes, governance considerations,
and materials that presuppose the existence of the original formulation.

本リポジトリは、判断透明性原則（Judgment Transparency Principle, JTP）そのものを
定義するものではありません。

JTPの原典的定式化は、SSRNに公開された学術論文として提示されています。
本リポジトリは、その原典の存在を前提として、
実装上の考察、ガバナンス上の注意点、関連資料を提供する
二次的参照点として機能します。
